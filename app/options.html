  1) Download and install Host.zip application, after extract run install.bat as Administrator
  2) Whitelist Chrome Extension by running postoolat.reg
  3) Connect printer and install Generic Text printer to USB000
  4) Add to your printing page meta tag with name "POSTOOLAT"
  5) From JS issue postoolat.print(html) or more safe window["postoolat_print"] = html;
  6) You can also issue native message to print see demo
